f(x) = (x - 5)(5x + 2) //is a quadratic function, and its graph is a parabola.
 
\begin{tikzpicture}
\begin{axis}[
  width=6cm,
  height=6cm,
  xmin=-10, xmax=10,
  ymin=-10, ymax=10,
  axis lines=middle,
  axis line style={blue!70, line width=1pt},
  grid=both,
  xtick distance=1,
  ytick distance=1,
  tick label style={font=\tiny},
  restrict y to domain*-10:10
]
\addplot[thick, red, domain=-10:10, samples=400] {(x-5)*(5*x+2)};
\end{axis}
\end{tikzpicture} 
 x - 5 = 0
 x = 5
 OR
 5x + 2 = 0
 5x = 0 - 2
 x/5 = -2/5
 x = -2/5
