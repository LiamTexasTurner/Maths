* parabola

** find x intercepts (y = 0)
[[http://www.desmos.com/calculator/gc71ymtspj][graph]]
f(x) = (x - 5)(5x + 2) //is a quadratic function, and its graph is a parabola.
 
\begin{tikzpicture}
\begin{axis}[
  width=6cm,
  height=6cm,
  xmin=-10, xmax=10,
  ymin=-10, ymax=10,
  axis lines=middle,
  axis line style={blue!70, line width=1pt},
  grid=both,
  xtick distance=1,
  ytick distance=1,
  tick label style={font=\tiny},
  restrict y to domain*-10:10
]
\addplot[thick, red, domain=-10:10, samples=400] {(x-5)*(5*x+2)};
\end{axis}
\end{tikzpicture} 
 x - 5 = 0
 x = 5
 OR
 5x + 2 = 0
 5x = 0 - 2
 x/5 = -2/5
 x = -2/5

\begin{tikzpicture}

\end{axis}
\end{tikzpicture}

 -1/3(x+2) = 0 [[file:../Algebra/Algebra.org::*Clearing the fraction ][how]]
 x + 2 = 0
 x = -2
 OR
 x + 8 = 0
 x = - 8
 
 midpoint = (-8 + -2) / 2 = -5

 find y when x = -5

 y = (-1/3)((-5+2)(-5+8))

 y = 3
